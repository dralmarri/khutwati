# خطوتي

تطبيق عربي RTL لمتابعة خفض الوزن من **82.7 كجم** إلى **77.5 كجم** بحلول **15 يوليو 2026**.

## المميزات

- خطة يومية مؤرخة تُبنى تلقائياً من يوم البدء حتى الموعد الذي يحدده المستخدم.
- جلسة يومية متكاملة تضم المشي والدراجة والمقاومة والتمدد.
- محرك تكيّف محلي يعدّل المدة والجولات بناءً على الإنجاز والجهد والألم المسجل.
- إعداد شخصي لوزن البداية والهدف والموعد ومدة الجلسة.
- اختيار مكان التدريب: المنزل أو النادي أو كلاهما.
- اختيار الأجهزة المتوفرة، مع دعم أجهزة النادي وتمارين وزن الجسم.
- واجهة مقسمة إلى صفحات مستقلة لتمرين اليوم والخطة الكاملة وسجل التقدم.
- شرح تفصيلي خطوة بخطوة لتمارين الحصيرة والكرة الطبية مع التكرارات والتنفس والتنبيهات.
- رابط فيديو تعليمي مستقل ملازم لشرح كل حركة من حركات الحصيرة والكرة الطبية.
- مؤقت مرحلي لكل جهاز يستمر أثناء تصفح التطبيق، مع بطاقة مصغرة وتحكم مستقل في مدة كل قسم.
- صفحة تغذية منخفضة النشويات مع حاسبة سعرات تقديرية وسجل يومي للوجبات والبروتين والنشويات والدهون.
- إدخال اختياري للعمر والحالات الصحية، مع احتياطات للكلى والغسيل والسكري والضغط والقلب والمفاصل.
- مدرب ذكي اختياري باشتراك شهري يحلل التقدم ويقترح تعديلات محافظة قبل تطبيقها بموافقة المستخدم.
- الخطة العادية والتسجيل والمؤقت والتغذية تبقى مجانية ولا تتطلب اشتراكاً.
- تصاعد تدريجي في مدة التمرين والجولات، ثم تخفيف قبل يوم الهدف.
- تمارين تستخدم جهاز المشي والدراجة والحصيرة وحبال المقاومة والكرة الطبية.
- صور وروابط فيديو تعليمية لكل نوع من التمارين.
- تسجيل الوزن وعرض سجل التقدم.
- متابعة الماء باللتر، بهدف إرشادي 2 لتر يومياً.
- مؤقت للتمرين وتسجيل إكمال تمرين اليوم.
- حفظ محلي باستخدام `localStorage`.
- تبقى البيانات محلية، ولا يُرسل ملخص التقدم إلى خادم الذكاء الاصطناعي إلا عند ضغط زر التحليل.
- دعم PWA والعمل دون اتصال.
- مشروع iOS جاهز لـXcode باستخدام Apache Cordova.

## تشغيل نسخة الويب

```bash
npm install
npm run web:serve
```

ثم افتح:

```text
http://localhost:4173
```

## تفعيل المدرب الذكي

انسخ `.env.example` إلى `.env`، ثم ضع مفتاح OpenAI في الخادم فقط:

```text
OPENAI_API_KEY=ضع_المفتاح_هنا
OPENAI_MODEL=gpt-5.5
ENTITLEMENT_SECRET=قيمة_سرية_طويلة
SUBSCRIPTION_VALIDATOR_URL=https://your-server.example/verify-apple-subscription
SUBSCRIPTION_VALIDATOR_TOKEN=رمز_الخادم_إن_لزم
```

ثم شغّل `npm run web:serve`. لا تضع المفتاح داخل ملفات `www` أو مشروع iOS. عند تشغيل تطبيق iOS يلزم نشر نقطة الخادم `/api/ai-plan` على عنوان HTTPS، ثم إدخال عنوانها في إعدادات التغذية داخل التطبيق.

الاقتراحات الرياضية والغذائية تقديرية، وليست تشخيصاً طبياً أو بديلاً عن الطبيب أو اختصاصي التغذية.

## اشتراك المدرب الذكي

معرّف منتج الاشتراك الشهري المستخدم في التطبيق:

```text
com.khutwati.ai.monthly
```

أنشئ في App Store Connect مجموعة اشتراكات ومنتجاً متجدداً تلقائياً بهذا المعرّف، ثم اختر السعر والدول المتاحة. التطبيق يجلب السعر والعملة من Apple، ويوفر الشراء واستعادة المشتريات.

خادم التحقق المحدد في `SUBSCRIPTION_VALIDATOR_URL` يجب أن يتحقق من معاملة Apple ويرجع:

```json
{
  "active": true,
  "productId": "com.khutwati.ai.monthly",
  "expiresAt": "2026-07-23T00:00:00.000Z"
}
```

لا يكفي الاعتماد على حالة الاشتراك المحلية في الهاتف؛ يتحقق خادم المدرب من استحقاق موقّع قبل تنفيذ طلب OpenAI.

## تجهيز iOS باستخدام Apache Cordova

يتطلب جهاز Mac وXcode.

```bash
npm install
npx cordova platform add ios
npx cordova prepare ios
npx cordova build ios
open platforms/ios/*.xcworkspace
```

إذا لم يوجد ملف workspace:

```bash
open platforms/ios/*.xcodeproj
```

## تجهيز Android لاحقاً

يتطلب Android Studio وAndroid SDK وJava.

```bash
npm install
npx cordova platform add android
npx cordova prepare android
npx cordova build android
```

لفتح المشروع في Android Studio:

```bash
open -a "Android Studio" platforms/android
```

مجلد الموقع الجاهز للنشر هو `www`.
